== Description ==

A quick look into REFLY :

	★ Publish message to Facebook, Twitter and LinkedIn with image
	★ Attach post or share link  to Facebook, Twitter and LinkedIn
	★ Publish and refly post with the publish and reply button
	★ Posts can be published and shared to social media with a single click.
	★ Enable or disable reply button for each posts

= REFLY Features in Detail =

Refly lets you publish posts automatically from your blog to Facebook, Twitter or Linkedin. You can publish your posts with title, description and images with single click. The plugin supports sharing content at the same time of publishing the wordpress post.

The prominent features of  the Refly plugin are highlighted below.

= Supported Mechanisms =

The various mechanisms of posting to Social Media are listed below. 

    Title of the post
    Description of the post
    Image scraped from the post
    Post to Profile or Page(Facebook, Linkedin, Twitter)
    Publish and Refly share your post at the same time your publish it.

= Filter Settings =

The plugin offers multiple kinds of filters for contents to be published automatically.

    Enable or disable refly button to share posts
    Share posts instantly

= Message Format Settings =

The supported post elements which can be published are given below.

    Post title 
    Post description
    Post Image
    Permalink
